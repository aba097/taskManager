# タスク管理アプリ
## タスク一覧（リスト形式）画面
- タスクの一覧が表示
- 検索ワードやカテゴリを絞って，検索することができる
- タスクを長押しすることで，どのカテゴリのタスクなのか・期限などの詳細な内容がAlertで表示される．また，タスクの編集も選択できる
<img width="363" alt="image" src="https://user-images.githubusercontent.com/26514249/194715751-f573b2b4-16d6-4b36-8693-85d19dddff7b.png">

## タスク一覧（カレンダー形式）画面
- カレンダー形式でタスクが表示
  - 日付の管理がしやすい
- タスクがある日の数字はタスクのカテゴリの色になる
- ここでもタスクの長押しが可能
  - 動作はリスト形式のときと同じ
<img width="363" alt="image" src="https://user-images.githubusercontent.com/26514249/194715904-c03280e0-0198-41d0-95f5-7cb08b749946.png">

## タスク追加画面
- タスクの追加が可能
- タイトルのみでも可
- タイトル，内容，日付（DeadLine），カテゴリを選択することができる
<img width="363" alt="image" src="https://user-images.githubusercontent.com/26514249/194715979-d22c81ef-6995-43b9-beaa-524d37a6bec4.png">

## カテゴリ選択画面
- カテゴリを選択することが可能
- カテゴリ名はTextFieldになっているため，好きなカテゴリ名に編集可能
  - カテゴリを選択のTextFieldは編集不可
- カテゴリの色をクリックするとColorPickerが立ち上がり，カテゴリの色を選択可能
- 新しいカテゴリが必ず1番下にあるため，好きなカテゴリを追加することが可能
